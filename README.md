This folder contains 26 zip files for Run 1 - Run 26, respectively.
Each zip file contains the source codes and data for all examples (i.e., Examples 1-58) using the HEESA algorithm.
In Runs 1-20, we did not call rng so that MATLAB automatically initialises the random number generator when the MATLAB session starts.
Runs 21-25 are performed using seeds 1–5, with the seed equal to the run number minus 20. 
sMU are implemented in GAMS24.6.1 and solved using the branch-and-bound solver CPLEX12 with default solver settings. 
All examples are solved on a desktop computer equipped with Intel® Core™ i7-10700 CPU @ 2.90 GHz, 32 GB RAM, running 64-bit Windows 11 Enterprise.
